## Deskripsi Project

**SIAKAD** adalah aplikasi Sistem Informasi Akademik berbasis web yang dibangun menggunakan **Laravel**. Aplikasi ini menyediakan fitur manajemen data akademik meliputi data jurusan, mahasiswa, dan mata kuliah, dilengkapi dengan autentikasi pengguna.

### Fitur Utama
- Autentikasi (Login & Logout)
- Manajemen Data Jurusan (CRUD)
- Manajemen Data Mahasiswa (CRUD)
- Manajemen Data Mata Kuliah (CRUD)
- Pencarian & Pagination pada setiap halaman data

### Teknologi yang Digunakan
- **Framework**: Laravel
- **Database**: MySQL
- **Frontend**: Bootstrap 5
- **Auth**: Laravel Auth
